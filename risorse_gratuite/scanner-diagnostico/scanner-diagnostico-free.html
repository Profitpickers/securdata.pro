Come si usa

1
Scarica e salva il file

Clicca il bottone "Download .bat" qui sopra. Salva il file in una cartella locale (es. C:\SecurData\). Non eseguirlo direttamente dalla cartella Download o da uno ZIP compresso.

2
Esegui come Amministratore (consigliato)

Clic destro sul file → "Esegui come amministratore". Se compare Windows Defender SmartScreen, clicca "Ulteriori informazioni" → "Esegui comunque". Funziona anche senza Admin, ma alcuni dati potrebbero essere parziali.

3
Attendi il completamento

L'analisi richiede 1–5 minuti (la sezione software è la più lenta). Non chiudere la finestra CMD durante l'esecuzione.

4
Leggi il tuo report

La cartella Report_Diagnostico_[data]_[ora] si apre automaticamente. Usa Notepad++ o il Blocco Note per una lettura più comoda dei file .txt.

File generati nel report

00_RIEPILOGO.txt

Indice di tutti i file con descrizione, data/ora analisi, computer e utente.

01_Hardware.txt

CPU (core, GHz, cache L2/L3), ogni banco RAM (velocità, produttore, slot), tutti i dischi con stato SMART, GPU (driver, VRAM), scheda madre + versione BIOS, batteria.

02_Sistema.txt

Versione e build Windows, uptime sistema, variabili d'ambiente, patch installate (KB).

03_Software.txt

Tutte le app installate (Win32 + Store) con versione e data, processi in esecuzione, servizi di sistema, task pianificati, driver installati.

04_Rete.txt

Configurazione IP completa, tabella ARP (IP→MAC), porte aperte e connessioni attive, DNS cache, cartelle condivise, routing, ping test Google.

05_Sicurezza.txt

Profili Firewall Windows, criteri di gruppo applicati, account utenti locali e gruppi Admin, audit policy, certificati utente, sessioni attive.

06_File_Sistema_Nascosti.txt

pagefile.sys / hiberfil.sys / swapfile.sys con dimensioni. Presenza e dimensione di Windows.old. Copie Shadow VSS. Flussi di Dati Alternativi (ADS) su file utente.

💡 Tips di lettura del report

  • → In 01_Hardware.txt sezione DISCHI: se Status=OK il disco è sano. Qualsiasi altro valore (es. "Pred Fail") → backup immediato!
  • → In 03_Software.txt → sezione APP: cerca software installato a tua insaputa (toolbars, trial scaduti, bloatware).
  • → In 02_Sistema.txt: le righe con "KB" numerati sono patch di sicurezza installate. Confronta con i bollettini Microsoft.
  • → In 06_File_Sistema_Nascosti.txt: se Windows.old è presente può pesare 20–40 GB. Usa il Toolkit PRO per rimuoverla in sicurezza.
